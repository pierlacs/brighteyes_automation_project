# BrightEyes — Company Overview

## About
BrightEyes is a startup providing diagnostic devices and services to ophthalmologists and opticians. Founded in 2023, based in Paris. Currently 18 employees.

## Products

### BrightEyes RetinaScan Pro
- Non-mydriatic retinal camera
- 14 megapixel CMOS sensor
- 60° field of capture
- AI-ready: automatic image quality assessment
- DICOM export, HL7 FHIR compatible
- **List price: €38,000**

### BrightEyes OCT-3000
- Spectral-domain OCT
- 100,000 A-scans/second
- Anterior segment module included
- AI-assisted retinal layer analysis (beta)
- Normative database with 5,000+ eyes
- DICOM export
- **List price: €62,000**

### BrightEyes AutoRef 500
- Automatic refractometer + keratometer
- Measurement time: 3 seconds
- Multilingual interface (EN, FR, IT, DE, ES)
- USB-C and Bluetooth connectivity
- **List price: €12,000**
